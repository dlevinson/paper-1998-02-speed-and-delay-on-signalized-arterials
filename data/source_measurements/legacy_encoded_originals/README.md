# Legacy Encoded Originals

This folder preserves the two old Unix `pack` source files from the 1987 Montgomery County speed/delay source set as one ZIP archive:

- `montgomery_county_1987_network_links_mclt87.Z`
- `montgomery_county_traffic_counts_mcnt.Z`

The loose `.Z` copies were removed from `original_legacy/` to avoid duplicating encoded and decoded variants throughout the package. Partial decode attempts and manifests are in `../legacy_z_decoded/`; the recovered outputs are incomplete and should be treated as forensic sidecars rather than complete data replacements.
