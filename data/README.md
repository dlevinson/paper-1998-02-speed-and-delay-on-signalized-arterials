# Data

This package contains the data for the 1998 Speed and Delay paper. The main analytical files are the 1987 Montgomery County link-observation workbook, the corresponding legacy Mac Systat dataset, and the publication table/regression/figure workbook. The `source_measurements/` folder adds the relevant upstream 1987 speed/delay source files from the Montgomery County `SPEED&DELAY` folder, including modernized CSV/XLSX sidecars where conversion was possible.
